# Edusistent — AI-powered образовательная платформа

## 🎯 Что это?

**Edusistent** — универсальная образовательная платформа с ИИ-тьютором, которая автоматически адаптируется под каждого студента. Мгновенная проверка заданий, персонализированные подсказки, адаптивная траектория обучения.

**Проблема:** Студенты ждут проверки заданий днями, не понимают свои пробелы, 90% бросают онлайн-курсы. Преподаватели тратят 70% времени на рутинную проверку.

**Решение:** ИИ проверяет задания за секунды, ведёт диалог как репетитор, строит индивидуальный план обучения.

## ✨ Ключевые фичи

### Для студентов:
- 📝 **Интерактивный редактор кода** — Monaco Editor с подсветкой синтаксиса
- ⚡ **Мгновенный фидбек** — код проверяется за секунды в изолированном контейнере
- 🤖 **ИИ-тьютор** — объясняет ошибки, даёт подсказки, не выдавая готовый ответ
- 🎮 **Геймификация** — XP, бейджи, лидерборды для мотивации
- 🌍 **3 языка** — узбекский, русский, английский

### Для преподавателей:
- 📊 **Аналитика группы** — видно слабые темы класса
- ✅ **Автогрейдинг** — освобождает от рутинной проверки
- 📚 **Генерация контента** — ИИ создаёт задачи по материалам курса

## 🛠️ Технологии

### Frontend (этот репозиторий):
- **Next.js 15** — React фреймворк с App Router
- **TypeScript** — типизация
- **Tailwind CSS + shadcn/ui** — UI компоненты
- **Framer Motion** — анимации
- **TanStack Query** — кеширование и синхронизация данных
- **Monaco Editor** — редактор кода (как в VS Code)
- **Zustand** — стейт менеджмент

### Backend (`qadam-backend`):
- **FastAPI** — асинхронный Python API
- **PostgreSQL** — база данных (14+ таблиц)
- **SQLAlchemy** — ORM
- **Docker** — изолированное исполнение кода
- **JWT** — авторизация
- **WebSocket** — real-time обновления

### LLM Service (`qadam-llm`):
- **LangChain** — оркестрация LLM
- **OpenAI GPT-4** — генерация подсказок и контента
- **RAG** — работа с учебными материалами
